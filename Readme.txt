A sleek, visually appealing personal portfolio website built with **HTML**, **CSS**, and **JavaScript**. Designed to showcase my skills, experience, and projects while providing an interactive and immersive user experience.

---

## Features

- **Background Animation**: A dynamic, glitch-inspired animation for an engaging visual effect.  
- **Neon Theme**: Eye-catching neon glow and pulse effects for text, buttons, and icons.  
- **Responsive Design**: Fully optimized for all devices, including desktop and mobile.  
- **Sections**:  
  - **Profile**: Introduction with glitch-enhanced image and title animation.  
  - **About**: Overview of my skills, experience, and education.  
  - **Experience**: Detailed breakdown of my expertise in frontend and backend development.  
  - **Projects**: Showcasing key projects with links to GitHub repositories and live demos.  
  - **Contact**: Easy access to my email and social media profiles.  

---

## Technologies Used

- **HTML5**: Semantic and structured content layout.  
- **CSS3**: Advanced animations, transitions, and responsive styling.  
- **JavaScript**: Interactive elements like the hamburger menu and section navigation.
